# ai-inmage
最近上映的《疯狂动物城》热度爆棚！今天想跟大家分享一个超级有创意的朋友圈晒照方式——用最新的 AI 技术与电影主角来一张“同框合影”。 你只需要上传指定的角色图片 + 一张自己的自拍，然后把下面的指令复制给豆包，就能 免费生成精致大片。快去试试吧！
指令如下（直接复制即可）：

不要换脸，人物脸部不能有任何改变！
请将上述图片中的人物与兔子、狐狸融合成一张三人俯拍自拍照，画面构图紧凑，三位主体靠得很近，兔子位于人物与狐狸之间，三人头部相贴，视线直视镜头，整体呈现强烈视觉冲击力。
左侧人物略微前倾并保持高度相似的面部特征，身体稍微内扣；拍摄角度为高角度俯拍，使头部比例被夸张放大，但不要拍到人物腿部，符合典型的日韩视觉自拍风格。画面需简洁干净、突出主体，风格偏日系视觉系，整体清晰度高，模拟 iPhone 前置摄像头自拍效果，呈现精致、时尚、略带氛围感的合影。
角色需无缝融入场景，光线自然、明亮均匀。背景为坐满观众的超大电影院，三人背对银幕，银幕上显示《疯狂动物城2》英文版精美海报（不要包含上映日期）。
